System requirements

To install and run this version of IAR Embedded Workbench you need:

  • A Pentium-compatible PC with Windows 7 or Windows 10, 64-bit versions.
  • At least 4 Gbyte of RAM, and 10 Gbytes of free disk space.
  • Adobe Acrobat Reader to access the product documentation

Third-party debugger drivers, might or might not work depending on their level of support for the Windows version used.

License management system

When you start the product for the first time, use the license number available in the welcome letter. Or, to evaluate the product, register with IAR Systems to get an evaluation license. In both cases you need to activate the license, which requires that your computer can connect to IAR Systems over the internet using port 443 (https).

If you have purchased the product, you will get a Startup license that is valid for 30 days. You will receive a notification by email when the permanent license is available. If you register for an evaluation license, you can get a 30-day time-limited evaluation license or a size-limited Kickstart license. For more information about installation and licensing, see below.
